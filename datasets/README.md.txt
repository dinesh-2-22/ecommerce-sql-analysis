# Ecommerce SQL Analysis Project

## Overview
This project analyzes an ecommerce dataset using MySQL to extract business insights related to sales, customers, and payments.

The goal of this project is to practice real-world SQL analytics used by Data Analysts.

---

## Dataset
Ecommerce orders and payments dataset containing:
- Order information
- Payment details
- Customer data

---

## Objectives
- Analyze yearly order trends
- Calculate total revenue
- Identify top customers
- Analyze payment methods
- Perform customer segmentation
- Classify customers based on spending behavior

---

## SQL Concepts Used
- SELECT statements
- JOIN operations
- GROUP BY & ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- CTE (WITH clause)
- CASE statements
- Window Functions
- Data Aggregation

---

## Key Insights
- Revenue increased year by year.
- Credit card payments generated the highest revenue.
- Majority of customers are low-value buyers.
- Only a small percentage of customers contribute to high revenue.
