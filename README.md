# E-Commerce SQL Analysis Project

## Project Overview
This project analyzes an e-commerce dataset using MySQL to generate business insights related to revenue, customers, and purchasing behavior.

The goal was to simulate real-world data analyst tasks using SQL.

---

## Dataset
Brazilian E-commerce Public Dataset (Olist)

Tables Used:
- olist_orders_dataset
- olist_order_payments_dataset

---

## Business Questions Solved

1. Yearly order growth
2. Revenue trend analysis
3. Payment method contribution
4. Top spending customers
5. Customer segmentation (Gold / Silver / Bronze)
6. Customer value segmentation (High / Medium / Low)
7. One-time vs Repeat customers

---

## SQL Concepts Used

- JOIN
- GROUP BY
- Aggregations (SUM, COUNT, AVG)
- CASE Statements
- Common Table Expressions (CTE)
- Window Functions
- Data Segmentation

---

## Key Insights

- Majority customers are low-value buyers.
- Credit cards generate highest revenue.
- Small percentage of customers contribute large revenue.
- Repeat customers form a smaller but valuable segment.
